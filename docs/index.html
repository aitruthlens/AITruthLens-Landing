TruthLens AI Detection API All Systems Active

The TruthLens AI Detection API lets you programmatically detect whether images, videos, or text are likely human-created or AI-generated using the same detection engine that powers the AI TruthLens browser extension and web app.

Base URL

https://api.aitruthlens.org/v1/

What You Can Do

  • Detect AI-generated images
  • Detect AI-generated video frames
  • Detect AI-generated text
  • Access unified JSON responses with provider + confidence scores
